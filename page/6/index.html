html笔记-基础

开始写之前

无论什么东西，快速了解都是有可能的，之后就要通过反复使用、反复操作获得并磨炼熟练程度。很多人被误导，总是以为“一定要学到一定程度才算学会，才可以拿出手……” 这是肤浅的。要尽快掌握“最少必要知识”（MAKE），然后就开始行动、开始践行…… 真正的熟练，只能在践行之中通过试错、改进、总结、纠正等必要的环节才能获得。

摘自《人人都是工程师》–李笑来

开始读之前

打开你的浏览器，点击右键，查看网页源代码，你会看到一堆看不懂的玩意儿，那就是html代码，这个系列笔记就是去简单地介绍html语言，并希望你们能去读懂它。给自己设个小目标吧，你希望从这个笔记中得到什么，是写代码还是能读懂就行？Anyway，定了目标，然后去做吧！

我们来举一个不太恰当的例子，简单解释一下网页。开始想象，打开局域网，你应该可以访问你的网上邻居(如果你有的话)的共享文件夹，打开文件夹，里面是他分享的一些文件，你可以浏览。
现在你把局域网换成外部网络，共享的文件夹当成网站，里面的文件就是网页，可你怎么看呢？浏览器就是你电脑上的文件浏览器。我们经常说把网站部署到server上，其实就是把文件在一台主机上共享出来。

OK ! Let’s start from here !

HTML基础 Tags

写HTML要牢记：

  • 所有HTML文档必须以<!DOCTYPE html>开头
  • 文档内以<html>开始，</html>结束
  • 牢记，浏览器展示出来的是<body>与</body>之间的内容！！！
  • 记得所有的tags都要小写

Headings

Headings就是浏览器的标签页上显示的名字

  • 标题有六种大小,从<h1>到<h6>
  • Headings很重要，它在一定程度上反映了html页面的结构，而Headings中的title则反映了页面的主要内容

  • 你可以在<head>与</head>之间定义你的title，甚至字符编码，就像这样：

  • <hr>则可以定义内容主题的改变（很拗口是吗？简单点，加条横线，换一页）

就会展示成如下：




This is heading 1


This is some text.

This is some other text.





Paragraphs

用<p>来展示段落正文，</p>来结束

<br>即break，暂时理解成换行吧，

注意！
在浏览器展示html页面时，它会自动忽略空格与缩进，这时<br>就很有用了

B！U!T!!!
HTML提供了一个很有用的tag，就是<pre>,preformatted–预格式化，<pre>元素内的文本以固定宽度的字体（通常为Courier）显示，并且它保留空格和换行符，（你基本上可以理解为，你写成什么样，展示出来什么样！）

Tags的Attributes(特性)

语言

标记在<html>中，如：<html lang="en-US">

这声明了你的语言类型，这对搜索引擎和展示页面来说，是非常重要的

标题

可以声明在你的正文内：<p title="I'm pretty">这样，你把鼠标放在那段正文上，会出现这个title

链接

链接用<a>标签，用法如下：<a href="http://www.baidu.com">This is baidu's link</a>
This is baidu’s link,你可以试试！

图片的尺寸

在<img>中去定义weight和height，

src 是source的意思，源文件

<img src="/asset/blogimg/dog.jpg" width="170" height="142">

<alt>标签，当无法显示图像时，alt属性指定要使用的替代文本。
属性的值可以由屏幕阅读器读取。 这样，某人“收听”网页，例如 一个盲人，可以“听到”的元素。

今天先写到这里，做一个小复习：

  • Headings有<h1>到<h6>标题大小,<hr>加横线，<title>写标题
  • PAragraph中<p>写正文，<br>换行，<pre>预格式化
  • 语言，标题，链接，图片尺寸几个Attributes
  • 别忘了，要小写！

Good luck to you ！

html第一课（翻译自官方文档）

什么是html

HTML 是用来写网页的一种标准标记语言。

H(Hyber) T(Text) M(Markup) L(language)
超文本标记语言
html用标记的形式来描述网页结构
html 元素是html页面的基本块（block，暂时想不起更好的翻译）
标签Tags代表了html的基本元素
内容的标签（label）叫做html tags
Web 浏览器的作用是读取 HTML 文档，并以网页的形式显示出它们。浏览器不会显示 HTML 标签，而是使用标签来解释页面的内容

My First Heading

<html>是html页面的基本要素
<head>包含了html文档的主要信息
<title>你的网页（网页实际上就是一篇文档）的标题
<body>文档的主要内容
<h1>标题的格式，h1是大标题，有h1到h6
<p>paragraph，正文

HTML 的Tags

  • 成对出现<> </>
  • 尖括号内填tag的内容，（tag实质上就是你定义了你的文档怎么去显示）
  • 结束的tag比开始时多了/,问号的那个斜杠

Web Browsers（网页浏览器）

之前说了，浏览器读取你的html文档，展示成网页，利用tags，但不把tags展示出来。

网页的结构

<!DOCTYPE > 是什么？

<!DOCTYPE>从文本是理解就是文档类型，它声明了这份document 的类型，比如<!DOCTYPE html>声明这是html文档，以便于浏览器正确的展示页面。

什么是Markdown？

Markdown是一种在网络上进行编辑的风格，你可以控制文档如何被展示，
字符加粗或斜体，添加图片，生成简单的列表等等，这些事都可以用Markdown来实现。
最重要的是，Markdown仅仅是普通的字符与一些特殊的字符的结合，比如#，*…
