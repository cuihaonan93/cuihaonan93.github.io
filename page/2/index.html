CSS笔记1

什么是CSS

在之前的html笔记系列中，提到过一点CSS，它的作用是用来定义html文档如何更优雅的展示在用户的浏览器上。

官方定义：

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

看得懂吧？

html笔记-表单1

html的表单

我单独把表单拿出来了，写一篇笔记，是因为我觉得重要，因为在几个月前，我在Atom实习时，需要做一个登陆界面，涉及到了表单（Form）的知识，我认为这个很重要，因此，单拿出来写。

什么是form

官方定义：

The HTML <form> element defines a form that is used to collect user input

回忆一下，你在网页上登陆你的微博，邮箱时，你输入你的用户名，密码等，这个就是表单，它能够捕捉到用户输入.。

fieldset

<fieldset>代码表示对标签区域内的表单数据进行分组，
<legend>的意思是<fieldset>的标题

action

<action>属性定义在提交表单时要执行的操作。比如，在提交完你的账号密码后，<action>会链接到一个js的代码，与数据库中已存的你的账号密码，去验证，如果符合，则登陆成功。

method

方法有“GET”和”POST”,我建议去搜一下，自己总结一下它们的不同。

form中的input

<input>是表单中最重要的元素，捕捉输入！
它的 <type> 属性有三种常见的方式：

  • <input type="text">
  • <input type="radio">
  • <input type="submit">

    text

    会出现一个文本框，你可以输入你的文本（默认20个字符）

    <textarea name="message" rows="10" cols="30">会定义一个多行的文本框

    radio

    会出现单选按钮（radio button），适用于可选项

    submit

    从字母意思可看出，会生成一个提交按钮

有两个重要的属性：
<name="">：把这个<input>中得到的信息看成一个变量，这就是变量名
<value="">而这个就是变量的值，在 <type="text">中，value就是默认的填好的值

怎么叫一个变量？

比如 <x = 5> x就是变量名，5是x默认的值

为什么要把我们的输入当成变量呢？

还是刚刚的例子，你的用户名密码，在输入之后，去数据库中对比，怎么找？怎么对比？根据 <username="你的用户名">去找你的信息，应该明白了吧？

html5中的方便标签

在html5中增加了许多的我们用起来很方便的type，实质上它们都是一些集成好function，你一定会看见你比较熟悉的type的。

数字

<input type="number" min="2" step="5">可以定义一个数字输入框，你可以加限制min,max,也可以指定步长step

颜色

<input type="color" name="favcolor" value="pink">
可以出现一个颜色选择框，默认选好是粉色

范围

<input type="range" name="points" min="0" max="10">可以出现一个滑动条，最小值是0，最大值是10。

日期

<input type="date" name="出行日期">
你一定买过火车票，在你选择出行日期时，那个包含年月日的选择框，就是这个

还有跟这个类似的type，比如：

  • <input type="month"> 这个只能选到月份
  • <input type="week">
  • <input type="time">这个呢，就是选一个时间，24小时制

邮件

<input type="email" name="email">这个就是自动检测你输入的邮件地址是否合法，比如有没有@，

链接

<input type="tel" name="主页">这个最简单了，检测你输入的url是否合法，比如我们在浏览器上改主页设置时，就会用到。

杂谈-驴得水

我将真心付给了你


我的好朋友爱看电影，我比较喜欢电视剧。

因为我的代入感总是很强烈，我总认为我就是剧中的一个人，跟着角色一起生活，一起开心，一起悲伤。但电影时间太短，我就会有“我只活了两个小时”的感觉，因此我更喜欢电视剧，尤其是长篇的，武林外传80集，我就会觉得自己活得久了些。

看电影，爱看搞笑片（注意，不是喜剧，是搞笑片），看一看嘻嘻哈哈，看一看吵吵闹闹。

这个毛病，一直保留到了现在。所以前几天，本想轻松一下，去电影院刷《驴得水》，结果发现我错了。

这里想提一句，《驴得水》在网上吵得火热，但我没看影评，所以才会错得那么彻底，以为真是一部喜剧…剧透的感觉，就像小时候攒了好久的零花钱，终于换到一根棒棒糖，准备带回家自己咂摸甜味时，一个小屁孩过来指着你的棒棒糖，牛逼哄哄得说：“昨天我妈刚给我买了，一点都不好吃”。

好吃不好吃不重要，探索的过程却变得索然无味。

驴得水的上半场，我几乎是睡过来的。主人公夸张的表情、舞台剧的荒诞表现方式、剧情衔接的断层，一切都太刻意。

直到一首歌进来……



还有老狼版本~


这首歌伴随着剧情的转变，我困意全无。此处就不剧透了，整整下半场，我的眼睛始终盯着大屏幕。

剧情结束，身边的朋友问：“这部电影是不是在讽刺男权主义？”

我白了他一眼，没说话。

他继续问：“你应该特别喜欢张一曼吧？”

我沉默了一会，回答：“对呀，她很性感，但她也很真实。”

在张一曼身上，有太多矛盾。剧中，大量素材不断暗示她是“婊子”的人物设定，丝毫不避讳谈性，镜头更是不掩饰。赤裸、咬耳朵、亲吻、娇嗔……玩弄、调教男人是家常便饭，她享受性，并以此为乐。

但镜头一转，张一曼身着素袍，坐在椅子上，秋日的阳光铺满一地的安静。她轻声哼唱《我要你》，清澈的音色飘散在空荡的原野，风悄悄地压弯了野草。眯起眼睛的她，将剥下的蒜皮一挥，幻成漫天雪花。

在这个女人身上，我看到了曾经说爱你的人不可得后的报复、当局者的利益被蚕食时从皮肤里渗出来的欲望、无能愤怒的人变成了听话的牲口、衣冠楚楚的人变成了双眼铮红的嗜血鬼，然而这些人活着好好的人，都“死”了。

电影中死去的张一曼，却是唯一“活”着的人。其他人呢？宽容的人，成了小人；淳朴的人，成了恶棍；正面向上的人，总在行骗。

活着的，已经死去，而死去的，却依然活着

导演一定是偏爱张一曼的，才会让她选择自己的结局，如此惨烈，如此动容。

电影结束后，我满脑子都是美国诗人罗伯特·弗罗斯特的墓志铭：“我和这个世界有过情人般的争吵。”

是啊，这样的女子…
