什么是html

HTML 是用来写网页的一种标准标记语言。

H(Hyber) T(Text) M(Markup) L(language)
超文本标记语言
html用标记的形式来描述网页结构
html 元素是html页面的基本块（block，暂时想不起更好的翻译）
标签Tags代表了html的基本元素
内容的标签（label）叫做html tags
Web 浏览器的作用是读取 HTML 文档，并以网页的形式显示出它们。浏览器不会显示 HTML 标签，而是使用标签来解释页面的内容

<html>是html页面的基本要素
<head>包含了html文档的主要信息
<title>你的网页（网页实际上就是一篇文档）的标题
<body>文档的主要内容
<h1>标题的格式，h1是大标题，有h1到h6
<p>paragraph，正文

HTML 的Tags

  • 成对出现<> </>
  • 尖括号内填tag的内容，（tag实质上就是你定义了你的文档怎么去显示）
  • 结束的tag比开始时多了/,问号的那个斜杠

Web Browsers（网页浏览器）

之前说了，浏览器读取你的html文档，展示成网页，利用tags，但不把tags展示出来。

网页的结构

<!DOCTYPE > 是什么？

<!DOCTYPE>从文本是理解就是文档类型，它声明了这份document 的类型，比如<!DOCTYPE html>声明这是html文档，以便于浏览器正确的展示页面。
