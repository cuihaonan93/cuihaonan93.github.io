问题2

python中的数据形式，其实这里我不想讲太多，因为在开头时已经说过，想把这个系列做的有意思，不要那么枯燥。
但是涉及到了，我就简单说一下。

学语言(无论是计算机语言也好，还是外语)，都要用已知来推未知，就是类比，学习的过程就是一直类比，并发现不同，从而不断扩展自己的知识面的过程。

因此，想，我们人类的数据类型都有什么，简单了，文字和数字。
基本是这两大类吧？
当然了，文字还可以再分类，数字也可以再分。但基本就是这样吧。
python基本也是这样，字符串呢，就把它想成文字，它的表示方法有三种：

  • 使用单引号(‘)
    • 用单引号括起来表示字符串，例如：
    • str='this is string';
  • 使用双引号(“)
    • 双引号中的字符串与单引号中的字符串用法完全相同，例如：
    • str="this is string";
  • 使用三引号(‘’’)
    • 利用三引号，表示多行的字符串，可以在三引号中自由的使用单引号和双引号，例如：
    • str=’’’this is string
      this is pythod string
      this is string’’’

数字呢，也有很多种，

  • 整数 int
  • 浮点数 float
  • 布尔型 True(1)/False(0)

我不想讲太多书上的东西，就说这么多吧，附个网址，自己看看！

现在介绍一个好方法叫find()

Python find() 方法检测字符串中是否包含子字符串 str ，如果指定 beg（开始） 和 end（结束） 范围，则检查是否包含在指定范围内，如果包含子字符串返回开始的索引值，否则返回-1。

所以呢，直接对origintext遍历一下，每一个字符都去translatebook中去find返回的位置就是我们要的位置，
