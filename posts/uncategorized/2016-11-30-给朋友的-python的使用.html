给朋友的--python的使用

《给朋友的–python的使用》

写在前面

前些天，在北京时，朋友建议我写一个python的简单入门教程，我答应了，也一直在思考如何下笔，一来是有些忙，学校里论文的事情；二来呢，没有一个成熟的思路，去完整的把一门语言的使用给架构起来。
但就像李笑来说的，用必要的，最少的知识先去做起来。今天呢，突然想起来有一个有趣的网站叫python-challenge,你可以点开看看，它是一个网页的过关游戏，不仅仅用python，java，C++等等都可以，就是你能破解了那个网页潜藏的意义，你就可以过关，它底下也有一点点提示。

我就想，可以以这种方式来写一些东西，即有趣又能实现一些功能。
因此，在这里，先给这个系列的笔记租一个大致的目标：

  • 尽量用不同的方法去完成这个挑战
  • 尽可能生动有趣的讲解python的使用
  • 尽量不涉及晦涩难懂的概念
  • 对于我，尽快地更新这个笔记

面对的目标群体也很明确：

  • 有兴趣的，不想当程序员的
  • 有点时间的

为什么说不想当程序员的？
因为我在写时会把学程序语言的顺序打乱，而且，为了能更好的理解，我或许会说出一些不恰当的例子，进而影响了你编程道路的发展。

python的实现

我们的任务是用python实现2的38次幂！

在这里，我默认了你的电脑里装了python，对了，Mac里好像是python2.7，我建议你用python3，因为与时俱进嘛！(如果不会，请自行谷歌)

创建一个python的文件，（很简单，如果你是Mac，试着学会去用Terminal，如果你是windows，试着用用命令行）

我的文件名叫’c1.py’

第二步

在”c1.py”中输入：print(2**38)
按 ctrl+R ，看看运行结果是什么？

结果：274877906944
python是脚本语言，可以这么理解，所见即所得，再加上python语言的集成度相当高，基本接近人类的自然语言（英语啊）。
所以，会一点英语，都可以看明白，print(2**38)–>打印2的38次幂
下面说python的运算符：（a=10,b=20)

操作符 中文 描述 例子
+ 加法 对操作符的两侧增加值 a + b = 30
- 减法 减去从左侧操作数右侧操作数 a - b = -10
* 乘法 相乘的运算符两侧的值 a * b = 200
/ 除 由右侧操作数除以左侧操作数 b / a = 2
% 模 由右侧操作数和余返回除以左侧操作数 b % a = 0
** 指数 执行对操作指数（幂）的计算 a**b = 10 的20幂
// 地板除 操作数的除法，其中结果是将小数点后的位数被除去的商。 9//2 = 4 而 9.0//2.0 = 4.0

非常简单，你可以自己试试。
