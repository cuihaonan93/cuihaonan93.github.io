什么是Markdown？

Markdown是一种在网络上进行编辑的风格，你可以控制文档如何被展示，
字符加粗或斜体，添加图片，生成简单的列表等等，这些事都可以用Markdown来实现。
最重要的是，Markdown仅仅是普通的字符与一些特殊的字符的结合，比如#，*…

Github风格(风味吗？:cry:)的Markdown

Github.com有自己风格的Markdown语法，这个版本提供了一些额外但是非常有用的特性，会让你在Github上编辑内容更方便。

注意，这些特性仅仅在Github上描述，评论，Pull Request时可用（毕竟这是Github版的MD！），比如@可用在SHA-1 hashes(哈希算法，加密的一种)、发布、请求等，Task list在Gist上可用来评论或写在Gist的md文件里等等。

强烈建议，不明白的自行google！

语法高亮

用三个波浪线那个按键+代码类型+代码+

任务清单

  • [x] @mentions, #refs, links, formatting, and tags supported
  • [x] list syntax required (any unordered or ordered list supported)
  • [x] this is a complete item
  • [ ] this is an incomplete item

建议把它用在你的第一条评论中，那样你就有了一个非常方便的关于你项目进度的指示器。
