js学习1

JavaScript

神奇的语言，用了JAVA的名儿，却和人家一点关系都没有。

什么是javascript

是浏览器端的html和web的编程语言，任何一个搞前段的都要学会。

什么是浏览器端？

把浏览器想象成一个操作系统，html就是那里的文档，web server就是主机，而浏览器就是我们的显示器，我们需要类似于C语言一样的编程语言，可以编程，把这个电脑的界面变得更酷炫！

大致是这样的，如有不对，恳请指出！

js可以干什么

主要针对的是html，所以，它能干的非常多，比如：

  • 改变页面的内容
  • 改变图片
  • 改变CSS
  • 可以把某个html元素隐藏起来

js写在哪

它和css一样，三种：

  • 放在 <head>里
  • 放在 <body>里
  • 外部文件，被引入，后缀名是xxx.js

js的输出

js与我们操作系统端的许多编程语言都不一样，它没有内建的output，比如，python，你可以
print("你好啊")，ruby，你可以puts "你好啊！"
但是JS没有，但它很神奇，有四个方法能间接的看到：

  • window.alert() 弹窗提醒
  • document.write() 重写html页面
  • innerHTML 改变html的某个元素（通过id匹配）
  • console.log() 控制台的log，在浏览器窗口按F12，你就明白了

js的语法

js是一种脚本语言，就是它会告诉浏览器做什么，然后被执行。
因此它的基本语法有五个：

Values, Operators, Expressions, Keywords, and Comments.

值，操作器（运算符），表达式，关键词，注释

对了，js以分号;为分隔符

定值可以是数字，也可以说是“string”（你要习惯这个说法）

var x x=66把66这个值赋给(=)了x这个变量(用var来声明)

js的数据类型

为什么要知道数据类型？那样处理起来数据会更加得心应手，js的数据一般有：数字，字符串，数组，对象等等（给你们英文：numbers, strings, arrays, objects and more）

var length = 16; // Number

var lastName = “Johnson”; // String

var cars = [“Saab”, “Volvo”, “BMW”]; // Array

var x = {firstName:”John”, lastName:”Doe”}; // Object

举个例子：

var x = 16 + 4 + "Volvo"; –>"20Volvo"

var x = "Volvo"+16 + 4 ;–>"Volvo164"
所以，在js中，同时处理字符串和数字时，是将数字变成字符串来处理的，就是这样：18–>"18"
