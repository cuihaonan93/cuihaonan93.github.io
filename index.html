python基本也是这样，字符串呢，就把它想成文字，它的表示方法有三种：

  • 使用单引号(‘)
    • 用单引号括起来表示字符串，例如：
    • str='this is string';
  • 使用双引号(“)
    • 双引号中的字符串与单引号中的字符串用法完全相同，例如：
    • str="this is string";
  • 使用三引号(‘’’)
    • 利用三引号，表示多行的字符串，可以在三引号中自由的使用单引号和双引号，例如：
    • str=’’’this is string
      this is pythod string
      this is string’’’

数字呢，也有很多种，

  • 整数 int
  • 浮点数 float
  • 布尔型 True(1)/False(0)

写在前面

前些天，在北京时，朋友建议我写一个python的简单入门教程，我答应了，也一直在思考如何下笔，一来是有些忙，学校里论文的事情；二来呢，没有一个成熟的思路，去完整的把一门语言的使用给架构起来。
但就像李笑来说的，用必要的，最少的知识先去做起来。今天呢，突然想起来有一个有趣的网站叫python-challenge,你可以点开看看，它是一个网页的过关游戏，不仅仅用python，java，C++等等都可以，就是你能破解了那个网页潜藏的意义，你就可以过关，它底下也有一点点提示。

我就想，可以以这种方式来写一些东西，即有趣又能实现一些功能。
因此，在这里，先给这个系列的笔记租一个大致的目标：

  • 尽量用不同的方法去完成这个挑战
  • 尽可能生动有趣的讲解python的使用
  • 尽量不涉及晦涩难懂的概念
  • 对于我，尽快地更新这个笔记

面对的目标群体也很明确：

  • 有兴趣的，不想当程序员的
  • 有点时间的

为什么说不想当程序员的？
因为我在写时会把学程序语言的顺序打乱，而且，为了能更好的理解，我或许会说出一些不恰当的例子，进而影响了你编程道路的发展。

什么是数组？

举个例子，你想买一些东西，都有：苹果，酸奶，卫生纸等等

你可以用 list来表示，但是这里有一个更好的，叫Array，数组，实际上就是一堆东西的集合，
比如像这样：var buy_things = ["apple","yogurt","toilet paper"]无论是看起来，还是整理起来都非常方便。

数组的方法

刚刚说了看起来和整理起来都非常方便，那我们分别来讨论一下“看”和“整理”的方法。

“看”的方法

看，实际上就是显示数组

两种方法显示的是一样的，因为默认数组展示是字符串，但如果你把数组中第一项改成数字，
然后 document.getElementById("demo2").innerHTML = typeof(fruits[0])
(这句话同样解释了怎么看数组的某一项)，会返回 number，明白了吧！

描述 方法
改变某一项的值 fruits[3]="Strawberry"第4项变成草莓

删除第一项 fruits.shift();香蕉会被删掉

删除、添加和切片

在Js中你可以用delete来删除数组中的某一项，但是，删完之后会留一个空（hole）在那，那个空是undefined，非常难看。
因此常常用 splice(2,0,"Lemon","Kiwi")这个函数：

The first parameter (2) defines the position where new elements should be changed (spliced in).
The second parameter (0) defines how many elements should be removed.
The rest of the parameters (“Lemon” , “Kiwi”) define the new elements to be added.这两个可以省略的！

切片是 slice(),项一把刀，你指哪切哪，只要剩下的。

留下限，不留上限，这样的形式：[low,up)
